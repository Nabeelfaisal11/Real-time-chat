# CSC 436 - Project 5: Real-Time Chat Application

This project is a real-time chat room built with Node.js, Express, and Socket.io.  
Users can enter a username, send messages, and see updates instantly from multiple clients.  
Messages and usernames appear in a modern UI with animations and a glass-style layout.
